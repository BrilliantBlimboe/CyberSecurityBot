CyberSecurityBotFinal
=====================

Description:
------------
CyberSecurityBotFinal is a C# console application that functions as an interactive chatbot focused on raising awareness about cybersecurity. The bot responds to user queries on topics like phishing, malware, ransomware, social engineering, VPNs, firewalls, antivirus software, and best practices for safe online behavior.

Features:
---------
- Interactive chat interface with a typing effect for responses.
- Plays a welcome audio message on startup.
- Displays ASCII art logo and greeting.
- Responds to a variety of cybersecurity-related keywords and phrases.
- Provides general cybersecurity tips.
- User can exit by typing 'quit'.

Supported Topics:
-----------------
- Phishing
- Malware
- Ransomware
- Social Engineering
- DDoS attacks
- Passwords & Authentication
- Firewalls
- VPNs
- Software updates & Patches
- Data backup
- Encryption
- Social media safety
- Public Wi-Fi safety
- Antivirus
- General cybersecurity tips

Installation:
-------------
1. Ensure you have .NET installed on your system.
2. Clone or download the repository.
3. Open the solution in Visual Studio.
4. Build the project to restore required resources.
5. Run the executable or start debugging from Visual Studio.

Usage:
------
1. Launch the program.
2. The bot will display a welcome ASCII art logo and play an audio greeting.
3. Type a message or question related to cybersecurity.
4. The bot will respond with helpful information.
5. To exit the chat, type 'quit'.

Example Commands:
-----------------
- hello / hi
- what is phishing
- tell me about ransomware
- cybersecurity tips
- backup
- vpn
- antivirus
- quit

Dependencies:
-------------
- System.Media (for audio playback)
- Threading (for typing effect)
- ASCIIArt class (included in project for logo and greetings)

Notes:
------
- Make sure the 'Resources' folder contains 'greeting.wav' for the audio greeting.
- Responses are based on keyword detection, so phrasing may affect results.
- This bot is for educational purposes and does not replace professional cybersecurity advice.
